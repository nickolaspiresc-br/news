# NewsAPI Fetcher

This project is my first deep dive into consuming **REST APIs** and handling **JSON** data using Python.

# 🚀 How to get started

Follow these steps to set up and run the project on your machine:


### 1. Clone the repository

git clone https://github.com/nickolaspiresc-br/news.git

### 2. API key

Create a file named .env in the root of the project.

Add your API key obtained from NewsAPI.org like this:
API_KEY=your_key

### 3. Run the script

main.py


⚠️ Important Notes (Free Plan Limitations):

Rate Limits: The free plan has a daily limit on the number of requests you can make.
Scope: The free plan is limited to specific search parameters (such as country=us).


🛠️ Technologies Used:

Python 3
Requests (for HTTP requests)
python-dotenv (for managing environment variables)
